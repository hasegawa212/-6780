## 操作手順

1. Slackワークスペースで対象ファイル（File ID をリンクから開く: `https://files.slack.com/files-pri/T.../F{ID}`）を開く → 「保存」or「ダウンロード」
2. ローカルからDriveの**テストフォルダ**へドラッグ&ドロップ
   - フォルダURL: `https://drive.google.com/drive/folders/1WB2gXvpgzjiI8kXzR2pZb0DRiVUQw9iK`
3. リネーム規則に従ってファイル名を変更:
   - 形式: `【Slack取得】{入金|出金}_{取引先名}_{元ファイル名（拡張子なし）}_F{Slack File ID後半5桁}.{拡張子}`

## 入金/出金の判定基準

- 取引先名が transactions.json の `income[].payer` に一致 → **入金**
- 取引先名が transactions.json の `outcome[].payee` に一致 → **出金**
- ファイル名に「AB間」「AB」を含む → **出金**（仕入側）
- ファイル名に「BC間」「BC」を含む → **入金**（販売側）
- 両方該当する場合: ファイル名のAB/BCマーカー優先、なければ入金優先

- 本番フォルダ（1.入金資料 / 2.出金資料）への配置は **HIKARU様の最終承認後** に行ってください。
