# RECRUIT — 人材募集ホームページ

人材を増やすための採用サイトの一式(静的HTML/CSS/JS)です。
ローカルでは `index.html` をブラウザで開くだけで動作します。

## ページ構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | トップ(ヒーロー、メッセージ、数字、選ばれる理由、注目職種) |
| `about.html` | 会社紹介(ミッション/ビジョン/バリュー、会社概要、沿革) |
| `jobs.html` | 募集職種一覧(エンジニア、デザイナー、営業、CS、マーケ、コーポレート、新卒) |
| `benefits.html` | 福利厚生・働き方 |
| `interview.html` | 社員インタビュー |
| `faq.html` | よくあるご質問 |
| `entry.html` | エントリーフォーム(クライアントサイドバリデーション付) |
| `privacy.html` | プライバシーポリシー |

## アセット

- `css/style.css` — レスポンシブ対応のメインスタイル
- `js/main.js` — ハンバーガーメニュー / スクロール演出 / フォーム検証

## カスタマイズ

実運用にあたっては、以下を自社の情報に差し替えてください。

- 会社名、住所、連絡先(各ページのフッター/`about.html`/`privacy.html`)
- 募集職種、給与レンジ、応募資格(`jobs.html`)
- 社員インタビューの内容と写真(`interview.html`)
- ロゴ(`index.html` ほか各ページ `.logo`)
- エントリーフォームの送信先(現状はクライアント側で完結。サーバー連携や Google フォーム等への切替を想定)

## デプロイ

静的ファイルのみなので、Netlify / Vercel / Cloudflare Pages / GitHub Pages
など任意のホスティングにそのままアップロードできます。
