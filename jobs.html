ENGINEER

フロントエンドエンジニア

雇用形態: 正社員 勤務地: 東京 / リモート可 年収: 500万〜900万円

仕事内容

  • 自社SaaSプロダクトのフロントエンド設計・実装
  • UI/UXデザイナーと連携した画面開発
  • パフォーマンス改善、アクセシビリティ向上
  • 新規機能の技術選定・プロトタイピング

必須スキル

  • HTML / CSS / JavaScript (TypeScript) の実務経験 3年以上
  • React、Vue、Svelte などモダンフレームワークでの開発経験

歓迎スキル

  • Next.js / Nuxt 等のフルスタックフレームワーク経験
  • パフォーマンスチューニング、Web Vitals 改善経験
この職種に応募する
