# 案件マスタ / 入出力スキーマ

BC 自動生成パイプラインの入出力定義。

## 1. `/extract` の出力 = `/generate` の入力 `ab`（AB重説の構造化JSON）

AB 側重要事項説明書から抽出した構造化データ。主なキー（`juyojiko_schema.Juyojiko`）:

| キー | 内容 |
|------|------|
| `bukken_type` | `戸建` / `区分` |
| `torihiki_taiyo` | 取引態様 |
| `gyosha` / `torikiishi` | 宅建業者・取引士（AB 側） |
| `urinushi` | 売主（A）の表示 `{address,name,biko}` |
| `fudosan` | 不動産の表示（戸建: `tochi`/`tatemono`、区分: `ittou_*`/`senyuu`/`shikichiken`） |
| `touki_meigi` | 登記名義人 |
| `senyuusha_uchi` | 第三者占有（賃借人）の有無・概要 |
| `horei` | 法令制限（`kuiki_kubun`,`yoto`,`nijuni_jo`,`kenpei`,`yoseki`,`doro`,`other_horei` 等） |
| `setsubi` | 飲用水・電気・ガス・排水の概要 |
| `kanri` | 区分管理（管理費・修繕積立金・管理組合・委託先・用途/ペット制限） |
| `joken` | 取引条件（`baibai_daikin`,`tetsuke`,`seisan_kisanbi`,`iyakukin_wariai` 等） |
| `yonin_jiko` | 容認事項（配列） |
| `tokuyaku` | 特約事項（配列） |

> 物件事実（`fudosan`,`touki_meigi`,`horei`,`setsubi`,`kanri`）は BC へ**そのまま引き継ぐ**。

### 用途地域 `horei.yoto` の選択肢（都市計画法第8条 / 重説様式）

第1種低層住居専用地域 / 第2種低層住居専用地域 / 第1種中高層住居専用地域 /
第2種中高層住居専用地域 / 第1種住居地域 / 第2種住居地域 / 準住居地域 /
田園住居地域 / 近隣商業地域 / 商業地域 / 準工業地域 / 工業地域 / 工業専用地域 /
用途地域の指定なし

略称（例 `第1種中高層`）でも可。`bc_schema.normalize_yoto()` が正式名称へ寄せる。

## 2. `/generate` の入力 `deal_master`（案件マスタ＝BC側の確定値）

Google Sheets「案件マスタ」タブ（`1bDAGArxrGwKQbY8F-IZ0RcfWaoPtRB7_F9UowPyEgig`）の列名を
これに揃える。

| フィールド | 型 | 説明 |
|------------|----|------|
| `buyer_C` | string | 買主C（最終買主）氏名 |
| `buyer_C_address` | string | 買主C 住所 |
| `bc_baibai_daikin` | integer | BC売買代金（円） |
| `bc_shohizei` | integer | うち消費税等相当額（任意） |
| `bc_tetsuke` | integer | 手付金（任意） |
| `bc_seisan_kisanbi` | string | 公租公課の清算起算日（任意） |
| `bc_torihiki_taiyo` | string | 取引態様（既定「売買 ・ 媒介」） |
| `seller_B` | string | 売主B 氏名（既定「株式会社Martial Arts」） |
| `seller_B_address` | string | 売主B 住所 |
| `bc_gyosha_*` | — | BC側媒介業者（`_menkyo_no`,`_menkyo_date`,`_shozai`,`_tel`,`_shomei`,`_daihyo`） |
| `bc_torikiishi_*` | — | BC側取引士（`_toroku_no`,`_shimei`,`_jimusho`,`_jimusho_shozai`,`_tel`） |

> `bc_gyosha_*` / `bc_torikiishi_*` が無ければ重説の業者・取引士欄は空欄で出力（手書き記入用）。

## 3. 旧方式（手順書 curl 互換）の最小入力

`{"bukken","extracted":{shozai,kuiki,yoto,nijuni_jo,kenpei,yoseki},"deal_master":{buyer_C,bc_baibai_daikin}}`
を渡すと、最小フィールドから BC重説を生成する（`ab` を組み立てる内部マッピングあり）。

## 4. `/generate` の出力

`{"filename": "...", "bukken": "戸建|区分", "xlsx_base64": "..."}`
（BC重要事項説明書の .xlsx を base64 で返す）。
